Kamu adalah seorang pengembang web pemula yang sedang membantu sebuah toko buku online "BukuKita" untuk membuat halaman web sederhana. Pemilik toko ingin memiliki halaman utama yang menampilkan judul toko, daftar kategori buku, serta beberapa buku unggulan dengan gambar dan deskripsi singkat.

Agar tampilan lebih menarik dan responsif, pemilik toko meminta kamu untuk menggunakan CSS untuk mempercantik tampilan halaman.

ğŸ¯ Tugas Kamu:

Buat sebuah halaman web statis index.html dan style.css yang menampilkan:

  1. Judul Halaman & Header

    • Tampilkan judul toko buku "BukuKita - Toko Buku Online" di dalam elemen header.
    • Buat menu navigasi sederhana dengan 3 tautan: Beranda, Kategori, Kontak.

  2. Daftar Kategori Buku

    • Buat daftar kategori buku menggunakan list (<ul> atau <ol>), dengan minimal 3 kategori, misalnya:
      • Novel
      • Komik
      • Teknologi

  3. Buku Unggulan

    • Gunakan CSS Grid atau Flexbox untuk menampilkan minimal 3 buku unggulan.
    • Setiap buku memiliki gambar, judul, dan deskripsi singkat.

  4. Tampilan yang Menarik

    • Gunakan CSS eksternal (style.css) untuk mengatur tata letak dan warna.
    • Berikan efek hover pada menu navigasi dan gambar buku.
    • Gunakan warna latar belakang yang nyaman untuk membaca.
